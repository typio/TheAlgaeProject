The Algae Project

The goal of this website is to demonstrate the process of growing algae, and is related to a paper I wrote on using algae as a fuel source in cars. I chose this topic because of it's increasing relevance as fossil fuel sources are depleted. This project relates to the global context of "The Biological Revolution". My plan was to encourage people to learn more about substitutes to oil, as well as advocating for energy producers to switch to sustainable fuels. The product is an interactive demo I made in JavaScript, to demonstrate the process of growing algae. Since starting this project I've learned about using algae as a biofuel, but also about making static websites with JS.
